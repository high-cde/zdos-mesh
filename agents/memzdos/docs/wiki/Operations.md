# Operations

## Manutenzione

Mantieni il fork aggiornato confrontando periodicamente il branch principale con l'upstream. Ogni modifica locale dovrebbe essere descritta in commit atomici e documentata nel README o in `docs/`.

| Attività | Frequenza consigliata |
|---|---|
| Verifica upstream | Periodica |
| Revisione dipendenze | Prima di ogni utilizzo operativo |
| Aggiornamento documentazione | A ogni modifica funzionale |
