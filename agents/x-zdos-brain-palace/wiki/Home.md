## 🚀 Quick Start

### Installation
```bash
git clone https://github.com/high-cde/x-zdos-quantum-ghostnet-os.git
cd x-zdos-backend
npm install
```

### Initialize System
```javascript
import { ZDOSBrainPalace } from './src/brain/index.js';

const palace = new ZDOSBrainPalace({
  maxWings: 100,
  maxAgents: 50,
  compressionEnabled: true,
  autoConsolidate: true
});
```

### Process a Thought
```javascript
const thought = await palace.think(
  'What is the relationship between learning and memory?'
);
```

## 🎓 Core Concepts

### Memory Palace
A hierarchical memory organization inspired by the classical Method of Loci. Data flows through:
1. **Wings** - Macro-memory regions (6 core)
2. **Rooms** - Specialized domains
3. **Closets** - Compressed storage (30x)
4. **Drawers** - Verbatim critical data

### Knowledge Graph
Temporal entity-relationship triples enabling:
- Multi-hop inference (up to 5 levels)
- Transitive reasoning
- Causal chain discovery
- Analogical reasoning

### Multi-Agent System
4 system agents + unlimited custom agents:
- **Reasoner** - Logical inference
