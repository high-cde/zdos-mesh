## 🚀 Quick Start

### Installation
```bash
git clone https://github.com/high-cde/x-zdos-brain-palace.git
cd x-zdos-brain-palace
npm install
```

### Run Tests
```bash
npm test
# Expected: 21/21 tests passing (100%)
```

### Start System
```bash
npm start
# Server running on port 3001
```

### Initialize Palace
```javascript
import { ZDOSBrainPalace } from './brain/index.js';

const palace = new ZDOSBrainPalace({
  maxWings: 100,
  maxAgents: 50,
  compressionEnabled: true,
  autoConsolidate: true
});

await palace.initialize();
```
