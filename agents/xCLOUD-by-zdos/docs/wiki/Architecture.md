# Architettura

xCLOUD Enterprise — Fork evoluto del Bitrix24 Developer Hub, completamente rebrandizzato e ottimizzato per l’ecosistema CloudX × ZDOS. Include DSN Layer, Portal Fusion, Nexus, Terminal e pipeline enterprise-ready.

Il repository deve essere mantenuto come modulo comprensibile e tracciabile dell'ecosistema ZDOS / Z-GENESIS. Ogni componente dovrebbe avere responsabilità esplicite, confini chiari e dipendenze documentate.
